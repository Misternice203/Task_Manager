# Task_Manager
Basic Task manager made using Python in VS Code

## Code (How to use) 
* Make sure you have Python 3.14 downloaded
* Download raw file from github labeled Main.py
* Open raw file in Python to use Task manager program
* Opem file in VS Code to look at code

## Overview
This project is a simple CLI Task manager that allows the user to create, view, delete, and complete tasks. So far it uses a list to store information related to the users input. In the future i will be adding a file storage to this project. i will also be adding Due Dates, Task priority, statistics based off of task information and more.

## Features
* Welcomes user to the Task manager
* Pulls up a "Menu"
* asks user what option they would like to complete
* Task name
* Due date
* Priority level
* Adds tasks to a list
* Lets user view tasks they have created
* Lets user Complete tasks
* Lets user Delete a task
* task numbering using enumerate() function for easier task selection
* Exits program if user selects
